# Otto

AsciiDoc static site generator.

# Quickstart

**Install Otto**

``` sh
gem install ottogen
```

**Initialize a new site**

``` sh
mkdir mysite && cd mysite
otto init
```

**Build the site**

``` sh
otto build
```

**Serve the site**

``` sh
otto serve
```

**View the site**

``` sh
open http://127.0.0.1:8778/
```
